# Prespecified discussion scenarios

These alternative paragraphs were written before final evaluation. They are not a menu for choosing a favorable narrative. After the checksum-frozen primary tables are reviewed, an applicable paragraph may be adapted only if it accurately reflects the prespecified contrasts, uncertainty, multiplicity adjustment, effect sizes, and clinical context. Inapplicable scenarios remain in the repository.

## Preprocessing-only pattern

If P1−P0 contrasts show consistent changes within both state levels while S1−S0 contrasts do not, the result would support an observation-boundary explanation within the simulator. Exact-timestamp SQI gating and the shorter staleness rule may have changed which causal BIS information reached the controller. Such a pattern would not establish that the threshold or staleness limit is universally optimal.

## State-only pattern

If S1−S0 contrasts change consistently within both preprocessing levels while P1−P0 contrasts do not, the result would support the relevance of recent dose, cumulative dose, and concentration features under the reconstructed dynamics. It would not show that all eight added variables are necessary, nor would it authorize post hoc feature selection.

## Both-factor pattern

If both sets of main-effect contrasts change in consistent directions and the interaction is limited, the two interventions may contribute separately under the study conditions. The magnitude and uncertainty of each within-level contrast should be reported; the four cell means alone are insufficient.

## Interaction-dominant pattern

If the interaction is prominent while component contrasts differ by the other factor's level, preprocessing and state representation should be discussed as jointly dependent design choices. A single pooled main-effect statement could obscure that dependency.

## No meaningful change pattern

If intervals and effect sizes do not support a practically meaningful change, the study would still bound the effects of these frozen interventions under equal compute. Absence of evidence in a one-seed reconstructed simulation should not be generalized to equivalence or clinical ineffectiveness.

## Accuracy–smoothness or utilization trade-off pattern

If BIS-error or range outcomes change in a favorable direction while propofol amount or action-change magnitude changes in an unfavorable direction, both consequences should be reported without collapsing them into a single ranking. The reverse trade-off should be treated symmetrically. Clinical relevance requires external criteria and cannot be determined by an unplanned composite score.
