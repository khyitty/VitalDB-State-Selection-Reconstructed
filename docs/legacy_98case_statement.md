# Legacy 98-case Statement

> This repository contains exploratory development based on a non-random
> first-100-case sample that yielded 98 usable cases. Its prediction and PPO results
> are not confirmatory full-cohort results. The test split was inspected during
> development, and the PK-PD/RL environment is a reconstruction from published
> information rather than an exact reproduction of the unpublished original
> implementation.

## Rule for this repository

The legacy 98 cases are excluded from the future confirmatory cohort. Legacy case
IDs, splits, scalers, imputers, Cp/Ce trajectories, feature rankings, selection
frequencies, candidate manifests, protocol hashes, cohort fingerprints, checkpoints,
metrics, figures, predictions, and PPO outputs must not be copied or consumed.

Legacy source code may be inventoried read-only. Any later migration requires an
exact source commit, explicit data-dependency classification, and independent tests.
