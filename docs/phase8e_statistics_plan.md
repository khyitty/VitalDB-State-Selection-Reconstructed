# Phase 8E Paired Statistics Plan

The analysis is a repeated-measures 2×2 design with preprocessing P0/P1 and state S0/S1. The same test case must contribute one explicit row for each condition before any contrast is computed.

Planned paired contrasts are P1S0−P0S0, P1S1−P0S1, P0S1−P0S0, P1S1−P1S0, and the interaction `(P1S1−P0S1)−(P1S0−P0S0)`. Summaries include mean and median paired differences, a subject-level paired bootstrap 95% interval, two-sided paired sign-flip permutation test, and paired standardized mean difference. Secondary outcomes use Holm multiplicity handling, following the existing Protocol v1.3 plan.

Only synthetic fixtures were evaluated during readiness preparation. No actual test-derived statistic, p-value, confidence interval, effect, or condition comparison was computed.
