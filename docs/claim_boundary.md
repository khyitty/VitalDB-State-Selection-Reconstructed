# Claim Boundary

This repository may report only that state representations were compared within a
nonclinical simulator reconstructed from public PK-PD equations and documented
settings.

It must not claim:

- exact reproduction of the unpublished original implementation;
- superiority to an unpublished paper implementation;
- clinical safety, dosing advice, or readiness for patient care;
- causal effects of a feature on BIS;
- generalization beyond the audited VitalDB cohort.

Engineering dry runs establish code behavior only. They are not cohort estimates,
quality-threshold evidence, prediction results, or control results.
